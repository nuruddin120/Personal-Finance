# অর্থসাথী — ইউজার আইডি লগইন ও সিংক সেটআপ (সহজ গাইড)

এতে আপনি একটি **ইউজার আইডি + পাসওয়ার্ড** দিয়ে লগইন করতে পারবেন, এবং যেকোনো ডিভাইস/ব্রাউজার থেকে ওই আইডি দিয়ে লগইন করলেই আপনার আপডেটেড সব তথ্য চলে আসবে।

> কেন একবার সেটআপ লাগে: ব্রাউজারে রাখা তথ্য নিজে থেকে অন্য ডিভাইসে যায় না — মাঝে একটা ফ্রি ক্লাউড (Firebase) লাগে যেখানে তথ্য নিরাপদে জমা থাকে। এই সেটআপ **একবারই** করতে হয়।
>
> ⚠️ লগইন/সিংক শুধু আপনার **প্রকাশিত (লাইভ) সাইটে** কাজ করবে — চ্যাটের প্রিভিউতে নয়।

---

## "Build" খুঁজে পাচ্ছেন না?
উপরের **সার্চ বারে (🔍)** নাম লিখে খুঁজুন — যেমন `Authentication`, পরে `Firestore`। "Build" মেনু খোঁজার দরকার নেই।

---

## ধাপ ১ — প্রজেক্ট তৈরি
1. যান: **https://console.firebase.google.com** → গুগল দিয়ে সাইন ইন।
2. **Create a project** → নাম দিন (যেমন `orthosathi`) → **Continue**।
3. Google Analytics সুইচ **বন্ধ** করে **Create project** → তৈরি হলে **Continue**।

## ধাপ ২ — লগইন চালু করুন
1. সার্চ বারে `Authentication` লিখে খুলুন → **Get started**।
2. **Email/Password** এ ক্লিক করুন → প্রথম সুইচ **Enable** → **Save**।
   *(এটি ভেতরে ভেতরে আইডি-পাসওয়ার্ড লগইন চালু করে; আপনি অ্যাপে শুধু ইউজার আইডি দেখবেন।)*

## ধাপ ৩ — ডেটাবেস তৈরি
1. সার্চ বারে `Firestore` → **Firestore Database** → **Create database**।
2. লোকেশন (যেমন **asia-south1**) → **Next** → **Start in production mode** → **Create**।

## ধাপ ৪ — নিরাপত্তা নিয়ম
1. Firestore পেজে **Rules** ট্যাব → সব মুছে নিচেরটা বসান → **Publish**:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /orthosathi/{uid} {
      allow read, write: if request.auth != null && request.auth.uid == uid;
    }
  }
}
```

## ধাপ ৫ — কনফিগ কপি
1. উপরে-বাঁয়ে ⚙️ → **Project settings**।
2. নিচে **Your apps** → ওয়েব আইকন **`</>`** → নাম দিন → **Register app**।
3. দেখানো `const firebaseConfig = { ... };` অংশটি **`{` থেকে `};` পর্যন্ত হুবহু কপি** করুন (হাতে কিছু বদলাতে হবে না)।

## ধাপ ৬ — অ্যাপে বসান ও আইডি তৈরি
1. অর্থসাথী (লাইভ সাইট) → মেনু → **"লগইন ও সিংক"**।
2. বড় ঘরে কনফিগ পেস্ট → **কনফিগ সংরক্ষণ করুন**।
3. একটি **ইউজার আইডি** (যেমন `rahim2026`) ও **পাসওয়ার্ড** (৬+ অক্ষর) দিন → **নতুন আইডি তৈরি করুন**।

🎉 হয়ে গেল! এখন থেকে তথ্য নিজে নিজেই ক্লাউডে সেভ হবে।

---

## অন্য ডিভাইস/ব্রাউজারে তথ্য পেতে
1. ওই ডিভাইসে সাইট খুলুন → **"লগইন ও সিংক"** → একই কনফিগ একবার বসান।
2. একই **ইউজার আইডি ও পাসওয়ার্ড** দিয়ে **লগইন করুন** → সব আপডেটেড তথ্য চলে আসবে।

## কিছু কাজ না করলে
- **কনফিগ পড়া গেল না** → `{` থেকে `};` পর্যন্ত পুরোটা বসিয়েছেন কিনা দেখুন।
- **লগইন হচ্ছে না** → ধাপ ২-এ Email/Password "Enable + Save" হয়েছে কিনা দেখুন।
- **তথ্য সেভ হচ্ছে না** → ধাপ ৩ ও ধাপ ৪ (Rules → Publish) ঠিক আছে কিনা দেখুন।
- **প্রিভিউতে হচ্ছে না** → স্বাভাবিক; লাইভ সাইটে চেষ্টা করুন।

## নিরাপত্তা টিপস
- আইডি ও পাসওয়ার্ড মনে রাখুন; পাসওয়ার্ড কাউকে দেবেন না।
- অ্যাপের পিন লক ও ব্যালেন্স লুকানো ফিচার চালু রাখুন।
